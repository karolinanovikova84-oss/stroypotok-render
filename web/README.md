# Web

Frontend на `Next.js` для новой системы строительной компании.

## Что уже есть

- главная страница
- страница объектов
- страница смен
- формы, которые обращаются к backend API

## Перед запуском

1. Создайте `.env.local` на основе `.env.local.example`
2. Убедитесь, что backend работает на `http://localhost:4000`
3. Установите зависимости:

```powershell
npm install
```

4. Запустите dev-сервер:

```powershell
npm run dev
```
